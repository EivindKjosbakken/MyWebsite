My warehouse simulation project:

This program is meant to simulate a warehouse, to see requirements for dimensions of the warehouse, number of robots required, how many items the warehouse can receive per timestep, etc. The programming is written completely from scratch. I first made a warehouse with blocks in it, each with different coordinates. Then made robots, and give them the ability to find storage cells that are free, and that the robot can you to and deliver off items. The robots can also receive orders consisting of several items, and determine which cell it needs to go to, so it can retrieve the items. The robot will then go to that storage cell. Everything happens automatically, when you want to run the simulation, you just enter the hyperparameters you want. hyperparameters you can choose from are: size of warehouse (with length in x and y direction), number of robots, number of different products (each shelf in a storage cell can only contain one type of product). Interval where items are sent to/orders are sent to warhouse. How many items are to be sent to the warhouse per interval, how many order of items are sent to warehouse per interval, if warehouse should be visually displayed (done with the tkinter package), and if information about warehouse should be printed to console. You then choose how many timesteps to execute, and it will run the simulation and print out results. (results will say for example for this size of warehouse, and this number of robots, the warehouse can take this many items per interval/orders per interval).

Since the visualisation of the warehouse is written in tkinter, I cannot display the warehouse in the browser, but I have linked to a youtube video showing how it works.
